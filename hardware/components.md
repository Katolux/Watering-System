# Hardware Components

The watering system is built from affordable, easy-to-find components.  
This file lists the main parts used in the project and their function.

---

## 🧠 Microcontroller

### **Arduino Uno R3**
- Main controller for the irrigation logic  
- Reads sensors and controls pump  
- Easy to program and widely supported

---

## 🪠 Watering Components
- **12V mini water pump** or **solenoid valve**  
- **4/7 mm irrigation tubing**  
- **Connectors, T-joints, drippers**  
- **12V power supply**  

---

## 📡 Connectivity
(Currently planned)
- ESP8266 or ESP32 module for API communication  
- Allows fetching weather forecasts from Open-Meteo

---

## 🔌 Power Management
- Separate 5V supply for Arduino  
- 12V supply for pump or valve  
- Optional buck converter if using a single source

---

## 🛠️ Tools Used
- Breadboard (early prototyping)  
- Soldering iron (permanent installation)  
- Wire strippers, electrical tape, heat shrink  

---

More detailed diagrams and wiring instructions can be found in the documentation folder.
